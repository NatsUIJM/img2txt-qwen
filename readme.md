1. 下载程序

```bash
git clone https://www.github.com/NatsUIJM/qwenOCRbyUIJM
```

2. 创建虚拟环境

```powershell
python -m venv .venv
```

3. 激活虚拟环境
    - Windows

    ```powershell
    .venv\Scripts\activate
    ```

    - macOS

    ```bash
    source .venv/bin/activate
    ```

4. 安装依赖

```bash
pip install -r requirements.txt
```

5. 运行程序

```bash
python process.py /path/to/your/file.pdf
```
